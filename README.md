# Portafolio Sergio Salas

Portafolio profesional desarrollado con ASP.NET Core MVC y .NET 10.

## Tecnologías

- ASP.NET Core MVC
- Razor Views
- C#
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Ejecutar localmente

1. Instala .NET 10 SDK.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta:

```bash
dotnet restore
dotnet run
```

4. Abre la URL mostrada en la consola.
